# 🖼️ CaptionAI: A Deep Learning Image Captioning System with TensorFlow and Keras

An AI-powered web application that automatically generates natural language captions for uploaded images using a custom-built deep learning model trained from scratch.

> **Note:** Captions generated by the model are not entirely accurate. The model was trained on Flickr8k and COCO datasets and performs best on everyday real-world photos. Niche, abstract, or unusual images may produce generic or partially incorrect captions.

---

## 🧠 How It Works

1. User uploads an image (JPG, PNG, WEBP)
2. OpenCV checks if the image is too blurry
3. EfficientNetB0 extracts image features
4. Transformer Encoder processes the features
5. Transformer Decoder generates the caption word by word
6. Caption is displayed with copy and download options

---

## 🛠️ Tech Stack

| Layer | Technology |
|---|---|
| Frontend | React + Vite |
| Backend | Python Flask 3.0.3 |
| ML Framework | TensorFlow 2.10.0 + Keras 2.10.0 |
| Feature Extractor | EfficientNetB0 (ImageNet pretrained) |
| Blur Detection | OpenCV (cv2) |

## 🚀 How to Run

### 1. Clone the repository
```bash
git clone https://github.com/karlowaowa/CaptionAI.git
cd CaptionAI
```

### 2. Set up the backend
```powershell
cd backend
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
python app.py
```
Wait for: `[INFO] All models loaded successfully!`

### 3. Set up the frontend
Open a new terminal:
```powershell
cd frontend
npm install
npm run dev
```

### 4. Open in browser
```
http://localhost:5173
```

## ✨ Features

- Drag and drop or browse image upload
- AI-generated caption using CNN + Transformer architecture
- Blur detection — flags low quality images automatically
- Image info display (filename, size, dimensions)
- Copy caption to clipboard
- Download caption as `.txt` file
- Caption history — last 5 generated captions per session
- Click any history item to view full image and caption
